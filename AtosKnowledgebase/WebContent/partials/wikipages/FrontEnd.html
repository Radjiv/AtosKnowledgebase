Voorbereiding

Als voorbereiding op het ontwikkelen van de front-end is het van belang een aantal dingen op orde te hebben. Hieronder vallen o.a. de juiste mappen structuur en het downloaden van de benodigde bestanden.

Mappen Structuur

Om het project overzichtelijk te houden voor jezelf en andere is het van groot belang een logisch indeling van je mappen te hebben. Voor dit project gebruiken we de volgende opzet:

  • css
  • images
  • partials
  • userstories
  • jscript
    • libs

Libraries

Na het opzetten van de mappen structuur gaan we de benodigde libraries downloaden. Binnen dit project gebruiken we AngularJS, jQuery en de Chosen.

Download de volgende bestanden:

  • AngularJS - https://angularjs.org/
  • AngularJS resource - https://code.angularjs.org/1.3.9/angular-resource.js
  • AngularJS route - https://code.angularjs.org/1.3.9/angular-route.js
  • JQuery - https://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js
  • Chosen - https://github.com/harvesthq/chosen/releases/download/v1.3.0/chosen_v1.3.0.zip
Na het downloaden plaats je de bestanden in JSCRIPT -> LIBS.

HTML pages

Het ontwikkelen van de website begint allemaal met het toevoegen van elementen op de pagina.

Index page

De index.html wordt onze hoofdpagina, deze komt dan ook direct onder de map WebContent.

Index.html aanmaken
maak het bestand index.html aan in de WebContent map.

  1. Definieer in het bestand de versie van HTML die je wilt gebruiken. In dit geval gaan we HTML5 gebruiken dus dan wordt het " <!DOCTYPE html> "
  2. Voeg vervolgens de html tag toe en definieer hier in de AngularJS module die wij straks gaan gebruiken met behulp van de "ng-app" tag, bijv: <html ng-app="MyApp">

Header
maak ene header tag aan en neem hier in op:

  1. Titel van je pagina:<title>Knowledgebase</title>
  2. Meta data, zoals welke charset gebruikt moet worden: <meta http-equiv="Content-Type" content="text/html" charset="utf-8" />
  3. Implementeren van libs bestanden, bijv: <script src="jscript/libs/angular.js"></script>

Body Maak een body tag aan en vul deze op met de gewenste content zoals bijvoorbeeld een navigatie menu, slider, footer, etc. Voor dit project hebben we naast overige tags een speciaal kader gedefinieerd voor onze AngularJS partials: <div id="content" ng-view>
Het AngularJS script zal de ng-view tag herkennen en zal onze partials hier in laden.

Partials
Nu het hoofdkader (index.html) is aangemaakt gaan we de subpagina's aanmaken.

  1. Ga naar de map partials maak hier een bestand home.html aan. Dit wordt onze homepagina.
  2. Maak nog meer partial html bestanden aan naar wens.
  3. Vul de partial html bestanden met de gewenste content.

AngularJS

Nu we de content vast hebben staan gaan we de functionaliteiten er aan toevoegen. Denk hierbij aan het inladen van de partials in de index.html, zoekfuncties of filters.

Hiervoor maken we drie javascript bestanden aan:
App.js - hier wordt onze AngularJS module aangemaakt en nemen we de routeprovider op.
Services.js - hierin worden onze services (GET, POST, etc.) opgenomen.
Controllers.js - dit bestand bevat alle functionaliteiten van de betreffende pagina.

App

Om ons AngularJS script werkende te krijgen moet er als eerst een module aangemaakt worden met een naam. De titel van de module komt overeen met de ng-app titel die we in onze index.html hebben opgenomen:
var app = angular.module('myApp', [ 'ngRoute','myApp.service' ]);

Vervolgens configureren wij de routeprovider van de module zodat elke pagina en controller aan de juiste URL wordt gekoppeld, zoals:
app.config(['$routeProvider', function ($routeProvider) { $routeProvider .when("/home", {templateUrl: "partials/home.html", controller: "PageCtrl"})

Controllers

Nu de module vaststaat en is geconfigureerd gaan we de controllers aanmaken die we in de app.js willen opvragen.

Maak op de onderstaande manier elke controller aan:
app.controller('PageCtrl', function ( $scope, $location, $http ) {
-plaats javascript code hier-
});

Hierbij vormt 'PageCtrl' de naam van de controller. Geef naast de naam ook de gewenste waardes mee aan de functie, in dit geval zijn dat de variabelen $scope, $location en $http. Plaats binnen de functie vervolgens de gewenste Javascript code.

CSS

Nu de inhoud en functionaliteiten vast staan moet het alleen nog in een mooie vorm gegoten worden, dit doen we met behulp van CSS.

Atos Stylesheet

Voor dit project maken we gebruik van Atos stylesheet zoals gebruikt op de website.

Met Google Chrome plukken we op de volgende manier de stylesheet van de Atos website:

  1. Ga naar http://atos.net/en-us/home.html
  2. Druk F12.
  3. Ga naar het tab Resources.
  4. Open links in de verkenner de map 'Frames', daarna '(home.html)', gevolgd door 'Stylesheets'.
  5. Hier zie je alle stylesheet bestanden, open deze en kopieer de content naar de Stylesheet map onder je WebContent.

Vervolgens openen we onze html pagina's. Naast deze leggen we de Atos html code en kennen we aan onze gelijksoortige html tags dezelfde id's en classes toe. Op deze manier nemen we de Atos style elementen eenvoudig over in ons eigen project zonder al te veel werk.
