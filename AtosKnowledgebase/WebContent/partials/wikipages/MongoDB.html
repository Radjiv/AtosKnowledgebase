MongoDB

Mongodb is een open-source document georiënteerde NoSql(geen relationeel databasemanagementsysteem) database. Er is geen schema, de documenten worden in de vorm van BSON (binair JSON) opgeslagen en de structuur van deze documenten is flexibel.

Installatie

Download hier de laatste versie van MongoDB: http://www.mongodb.org/downloads. Installeer deze in een gewenste map.

Doorloop vervolgens de volgende stappen:
  1. Open in je command propt mongod vanuit de bin map in de mongodb folder.
  2. Zodra mongo voor het eerst wordt gebruikt, specificeer dan eerst de default folder met het command: md \data\db
  3. Open hierna vanuit een tweede command prompt mongo vanuit de zelfde map.
  4. Zodra mongo is gestart gebruiken we het atos database met het command: use atosdb.
  5. Bevestig hierna met het command db dat je deze database dat werkelijk gaat gebruiken.
  6. Mongo is klaar voor gebruik.

Tip: Hernoem de mongodb folder naar mongodb. Hierdoor hoef je niet elke keer bij het opstarten van de database de gehele folernaam in de command prompt te typen. Een ander mogelijkheid is om de map te openen in explorer, vervolgens shift ingedrukt houden en met de andere muisknop klikken. Kies hierna Open command window here. Hierdoor is de command promp gelijk in de juiste map.

Werking

Mongodb is programmatisch verwerkt in de applicatie doordat er een User class is aangemaakt. Deze User class omvat de volgende variabelen:

  1. id (String)
  2. firstName (String)
  3. middleName (String)
  4. surName (String)
  5. dateOfBirth (String)
  6. function (String)
  7. competence (List<String>)
  8. vip (boolean)

Deze User class is ge-inject met het Morphia framework. Het Morphia framework maakt een koppeling tussen de User class en de DAO. Door deze combinatie wordt door het aanroepen van DAO methoden de database ge-update.

Zodra de server is gestart kan op de website Users toegevoegd, gewijzigd of verwijdert worden. Vanuit de mongo command prompt, kan middels het command: db.User.find() de inhoud van de database opgevraagd worden.
